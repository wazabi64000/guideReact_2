02 — Structure de projet & Premier composant

Objectif (fil rouge)

Poser la structure du projet pour l'application Animaux et coder le composant AnimalCard qui affichera les données d'un animal.

Structure recommandée

src/
├─ main.jsx          // branche l'application dans index.html
├─ App.jsx           // composant racine — contiendra la logique globale
├─ assets/
├─ data/
│  └─ animaux.json   // source de données locale pour l'exercice
└─ components/
   └─ AnimalCard.jsx // composant réutilisable pour afficher un animal

Exemple : fichier src/data/animaux.json

[
  { "id": 1, "nom": "Chat", "age": 3 },
  { "id": 2, "nom": "Chien", "age": 5 },
  { "id": 3, "nom": "Lapin", "age": 2 }
]

Exemple : composant AnimalCard.jsx

// src/components/AnimalCard.jsx
export default function AnimalCard({ nom, age }) {
  return (
    <article className="animal-card">
      <h3>{nom}</h3>
      <p>Âge : {age} ans</p>
    </article>
  );
}

Exemple : point d'entrée src/main.jsx

// src/main.jsx
import React from 'react';
import { createRoot } from 'react-dom/client';
import App from './App';

createRoot(document.getElementById('root')).render(<React.StrictMode><App /></React.StrictMode>);

Exemple : src/App.jsx minimal pour afficher la carte

// src/App.jsx
import AnimalCard from './components/AnimalCard';

export default function App() {
  return (
    <main>
      <h1>App Animaux — Étape 1</h1>
      <AnimalCard nom="Chat" age={3} />
    </main>
  );
}
